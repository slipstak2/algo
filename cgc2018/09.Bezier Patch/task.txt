Реализовать построение объемных фигур, состоящих из бикубических поверхностей Безье. Координаты опорных точек заданы текстовым файлом. Формат входного файла: 1-я строка содержит число N - количество бикубических поверхностей Безье в фигуре. Далее идут N фигур, заданных координатами (X,Y,Z) 16-ти опорных точек (в текстовом виде). Примеры файлов моделей прилагаются к заданию. Визуализацию допустимо выполнить в каркасном (wireframe) виде с использованием предыдущих заданий (построение отрезков прямой, преобразования координат и математическая библиотека и т.п.). Желательно использовать для визуализации центральную проекцию.

Общие пожелания к сдаваемым работам

Работы желательно присылать в ZIP архиве. Архив состоит из трех директориев: BIN, SRC и SHOTS. В директорий SRC разместить целиком дерево проекта в той среде, в которой выполнялось задание, в директорий BIN - исполняемый файл (если Вы выполняли задание под MS-Windows) со всеми вспомогательными файлами данных и, возможно, динамических библиотек, SHOTS - желательно сделать screenshot-ы экранов (окна) работы программы. Поясняющие комментарии в исходных текстах приветствуются. При необходимости можно разместить в корне архива файл INFO.TXT с пояснениями к выполненному заданию.
